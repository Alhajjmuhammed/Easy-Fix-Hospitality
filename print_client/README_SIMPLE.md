## What You Need
- Windows Computer
- Thermal Printer (USB connected)
- Internet Connection

---

## STEP 1: Install Python

1. Go to: **https://www.python.org/downloads/**
2. Click **"Download Python"** (yellow button)
3. Run the installer
4. ‚ö†Ô∏è **CHECK THE BOX: "Add Python to PATH"** ‚Üê VERY IMPORTANT!
5. Click **Install Now**
6. Wait, then click **Close**

---

## STEP 2: Run Setup

1. Double-click **SETUP.bat**
2. It will install everything automatically
3. When asked, enter your API Token:
   - Go to https://hospitality.easyfixsoft.com
   - Login with your username/password
   - Click **Printer Settings**
   - Copy the token and paste it
4. Press Enter

---

## STEP 3: Connect Printer

1. Connect printer to computer via USB
2. Turn ON the printer

---

## STEP 4: Start Printing

1. Double-click **start_print_client.bat**
2. Leave the window open all day
3. Orders will print automatically!

---

## Daily Use

**Every morning:**
1. Turn on computer
2. Turn on printer  
3. Double-click **start_print_client.bat**
4. Leave it running

---

## Troubleshooting

| Problem | Solution |
|---------|----------|
| "Python not recognized" | Reinstall Python, CHECK "Add to PATH" |
| "Cannot connect" | Check internet connection |
| "Authentication failed" | Get new token from Printer Settings |
| "Printer not found" | Check USB cable, restart printer |
